SVG — recomendado

Funciona en cualquier browser moderno. Fondo transparente. El loop es CSS nativo, sin JS. 
<img src="benefix-loader.svg" width="80" height="80" alt="Cargando">

Lottie JSON — para React Native / Flutter

Ideal si la app es mobile nativa. Usá lottie-react-native o lottie_flutter. En web, con lottie-web: 
import lottie from 'lottie-web';
lottie.loadAnimation({
  container: document.getElementById('loader'),
  path: '/benefix-loader.lottie.json',
  renderer: 'svg',
  loop: true,
  autoplay: true,
});

GIF transparente

No generé GIF porque se pixelariza y pesa 10–20× más que el SVG para el mismo resultado. Si lo necesitás: abrí benefix-loader.svg en ezgif.com/svg-to-gif, setealo a 200×200, 30fps, 3s, background transparent.
